# 🏫 School Result Portal

Multi-school platform for managing and viewing student results.

## Features
- 🏫 Multiple School Support
- 📊 Excel File Upload
- 👨‍🎓 Student Result Portal
- 📄 PDF Report Generation
- 🎯 Admin Dashboard
